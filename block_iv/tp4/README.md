# TP 4 — Paulo Queiroz

EventUp — plataforma de eventos comunitários construída com React, TypeScript e Cloudflare Workers.

## Deploy

https://bloco-iv-tp-4.s48jvgjnvp.workers.dev

## Comandos

```bash
bun dev          # servidor de desenvolvimento
bun test         # testes de componente (Vitest)
bun run cypress  # testes E2E (Cypress)
bun run build    # build de produção
bun run deploy   # deploy para Cloudflare Workers
```
